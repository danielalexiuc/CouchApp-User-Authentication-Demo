CouchApp-User-Authentication-Demo by danielalexiuc

Working demo of user authentication in a CouchApp. Requires CouchDB 0.11

Dependencies

CouchDB 0.11 CouchApp 0.5

Install

Install and start CouchDB 0.11 Install couchapp. Then, at your command prompt: couchapp push CouchApp-User-Authentication-Demo http://127.0.0.1:5984/demo
